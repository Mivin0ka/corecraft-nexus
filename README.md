# 🌌 CoreCraft Nexus — Control & Management Panel

**CoreCraft Nexus** — это легковесная, быстрая и полностью автономная веб-панель (SPA), разработанная специально для администраторов, разработчиков и технических специалистов игровых серверов Minecraft. Она объединяет в себе все необходимые инструменты для планирования задач, ведения документации, трекинга багов и мониторинга структуры плагинов.

---

## 🚀 Основные возможности

* **📊 Главная панель (Dashboard):** Краткая сводка по активным задачам, критическим багам, состоянию серверов и быстрому добавлению заметок.
* **📋 Менеджер задач (Tasks):** Удобный список дел с разделением по приоритетам (Низкий, Средний, Высокий) и категориям.
* **🐛 Трекер багов (Bug Tracker):** Полноценное логирование ошибок с указанием статуса (`New`, `In Progress`, `Testing`, `Resolved`), приоритета и шагов для воспроизведения.
* **📝 Заметки (Notes):** Быстрые текстовые записи и идеи с возможностью закрепления важных карточек.
* **🖥️ Архитектура серверов (Servers):** Наглядная структура вашей сети серверов (например, *Hub, Survival, SkyBlock, Anarchy*) с указанием портов, ядер и статуса.
* **🔌 Глобальные плагины (Global Plugins):** База данных используемых на серверах плагинов с версиями, типами лицензий и удобным поиском/фильтрацией.
* **🧪 Тестирование (Testing):** Чек-листы для проверки сборок перед релизом, логи тестов и фиксация результатов.
* **📖 Локальная база знаний (Wiki):** Статьи и шпаргалки по командам, конфигам или правилам для внутреннего использования персоналом.
* **🌗 Двухкомпонентный интерфейс:** Полноценная темная (`Dark`) и светлая (`Light`) темы, переключаемые в один клик.

---

## 🛠️ Технологический стек

Проект реализован в концепции **Single File App** (всё приложение внутри одного файла `index.html` без внешних тяжелых зависимостей):
* **HTML5** — семантическая структура и скелет приложения.
* **CSS3** — адаптивная верстка с использованием CSS-переменных (Custom Properties) для кастомизации тем. Шрифты: *Onest* и *JetBrains Mono*.
* **Vanilla JS (ES6+)** — чистый JavaScript для реализации всей бизнес-логики, рендеринга компонентов, фильтрации и управления состоянием без использования фреймворков.

---

## 💾 Хранение данных и безопасность

Приложение является **полностью клиентским (Client-Side)**. Это дает ряд важных преимуществ:
1.  **Конфиденциальность:** Данные вашего сервера не отправляются ни на какие сторонние сервера или базы данных.
2.  **Автономность:** Всё сохраняется локально в вашем браузере через `localStorage`.
3.  **Импорт / Экспорт:** В левом нижнем углу панели встроены функции резервного копирования. Вы можете в любой момент выгрузить свои данные в один `.json` файл и перенести их на другое устройство или поделиться с командой.

## 📦 Развертывание и использование

Так как проект не требует бэкенда, запустить его можно двумя способами:

1.  **Онлайн (GitHub Pages):** Просто перейдите по официальной ссылке: [corecraft-nexus](https://mivin0ka.github.io/corecraft-nexus/).
2.  **Локально:** Скачайте репозиторий, кликните дважды по файлу `index.html`, и панель готова к работе прямо с вашего компьютера.
